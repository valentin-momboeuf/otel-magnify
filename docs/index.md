# otel-magnify

Centralized OpenTelemetry agent management via [OpAMP](https://opentelemetry.io/docs/specs/opamp/).

This documentation is organized for three audiences:

- **[Users](users/index.md)** — deploy otel-magnify and connect agents
- **[Developers](developers/index.md)** — contribute to the codebase
- **[API](api/index.md)** — integrate with the REST, WebSocket, and OpAMP endpoints
